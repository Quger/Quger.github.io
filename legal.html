1. Введение

Настоящая Политика конфиденциальности описывает, как VoiceScribe («Бот», «мы») собирает, использует и защищает данные пользователей Discord при использовании сервиса транскрипции голосовых сообщений.

2. Какие данные мы обрабатываем

В процессе работы бот может обрабатывать следующие данные:

  • Голосовые сообщения и аудиофайлы, отправляемые для транскрипции;
  • Discord User ID и Server ID для идентификации запросов;
  • Метаданные сообщений (время отправки, идентификатор канала);
  • Технические логи работы для отладки и улучшения сервиса.

3. Как мы используем данные

Собранные данные используются исключительно для:

  • Транскрипции аудио в текст и возврата результата пользователю;
  • Улучшения качества работы сервиса;
  • Диагностики ошибок и технической поддержки;
  • Соблюдения требований Discord API и законодательства.

4. Хранение и удаление данных

Аудиофайлы не хранятся на серверах VoiceScribe дольше, чем требуется для обработки транскрипции. После формирования ответа аудиоданные удаляются автоматически.

Технические логи хранятся не более 30 дней, после чего удаляются.

5. Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам. Обработка аудио осуществляется с использованием API распознавания речи; при этом данные передаются только в зашифрованном виде и не хранятся провайдером сверх необходимого для выполнения запроса.

6. Безопасность

Мы применяем технические меры защиты (шифрование при передаче, контроль доступа) для предотвращения несанкционированного доступа к данным пользователей.

7. Ваши права

Вы вправе в любое время удалить бота с сервера и тем самым прекратить обработку данных. Для запроса на удаление конкретных данных обратитесь к нам по указанному контакту.

8. Изменения политики

1. Introduction

This Privacy Policy describes how VoiceScribe ("the Bot", "we") collects, uses, and protects Discord users' data when using the voice message transcription service.

2. Data We Process

During operation, the bot may process the following data:

  • Voice messages and audio files submitted for transcription;
  • Discord User ID and Server ID for request identification;
  • Message metadata (timestamp, channel ID);
  • Technical operation logs for debugging and service improvement.

3. How We Use Data

Collected data is used solely for:

  • Transcribing audio to text and returning the result to the user;
  • Improving service quality;
  • Diagnosing errors and providing technical support;
  • Complying with Discord API requirements and applicable law.

4. Data Retention and Deletion

Audio files are not stored on VoiceScribe servers longer than required to process the transcription. Once the response is generated, audio data is deleted automatically.

Technical logs are retained for no more than 30 days, after which they are deleted.

5. Third-Party Sharing

We do not sell or transfer your data to third parties. Audio processing uses a speech recognition API; data is transmitted in encrypted form only and is not retained by the provider beyond what is necessary to fulfill the request.

6. Security

We implement technical security measures (encryption in transit, access controls) to prevent unauthorised access to user data.

7. Your Rights

You may remove the bot from your server at any time to stop data processing. To request deletion of specific data, contact us via the details below.

8. Policy Changes

We reserve the right to update this policy. The current version is always available on this page.
